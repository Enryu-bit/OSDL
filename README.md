# OSDL
osdlmini - Hotel Management System

## Prerequisites
- Java 17 or higher
- Maven 3.6+

## How to Compile
You can compile the project using standard Maven commands:
```bash
mvn clean compile
```

## How to Run
Since this is a JavaFX application, you can easily run it via the JavaFX Maven plugin:
```bash
mvn javafx:run
```
